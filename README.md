# Hướng dẫn build
Tài liệu này sử dụng mkdocs để tự động generate static HTML page
## Cài đặt môi trường
- Cài đặt Python
- Install mkdocs ở local: pip install mkdocs
- Create new docs source project: tại thư mục muốn tạo, chạy cmd: mkdocs new GITHUB_PROJECT
